## Getting Started

  Clone and cd:
  ```
  git clone https://github.com/kroogs/prehensile cd prehensile
  ```

  Install dependencies:
  ```
  npm install
  ```

## Usage

  Watch `src` for changes:
  ```
  npm start
  ```

  Run tests:
  ```
  npm test
  ```

  Type check, reformat, lint:
  ```
  npm run lint
  ```

  Produce minified bundle in `dist`:
  ```
  npm run build
  ```
